# Data Entry Speed Trainer

Train for real data entry jobs. Build typing speed and accuracy across 10 job paths and 18 skill categories.

## Tech Stack
- React + Vite
- React Router v6
- Recharts
- bcryptjs
- localStorage (no backend)

## Features
- Account creation with bcrypt password hashing & recovery codes
- 10 job-specific training paths
- 18 skill drills (names, addresses, invoices, 10-key, etc.)
- Real-time character coloring during practice
- WPM, KPH, accuracy tracking
- XP system with 6 levels
- 40+ badges
- Daily challenge with streak tracking
- Progress charts (Recharts)
- Job readiness calculator
- Privacy-first: all data stays in your browser

## Development

```bash
npm install
npm run dev
```
